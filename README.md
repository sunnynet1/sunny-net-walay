# Sunny net walay - Static Deployment

This is the static version of the Sunny net walay ISP Admin Dashboard.

## Deployment Instructions (Vercel)

1. Ensure you have the following structure:
   - `index.html`
   - `vercel.json`
   - `assets/`
     - `css/style.css`
     - `js/app.js`
     - `images/`

2. Upload these files to a new Vercel project.
3. Vercel will automatically detect the `index.html` and serve it as a static site.

## Features
- **Dashboard**: Real-time analytics using Chart.js.
- **CSV Upload**: Process customer lists directly in the browser.
- **Local Storage**: Data persists in your browser's local storage.
- **Responsive Design**: Works on mobile and desktop.
